### Как какать в репозиторий?

Создать новую ветку:
```
git branch <name>  
```

[Опционально] Посмотреть список веток:
```
git branch --list
```

Переключиться на новую ветку:
```
git checkout <name>
```

Внести изменения и закомитить.  

  
Создать ветку на удаленном репозитории:
```
git push --set-upstream origin <name>
```

Запушить.
